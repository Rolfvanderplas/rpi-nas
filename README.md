# Raspberry PI NAS
Setup your own NAS on a Raspberry Pi

## Prerequisite
- Docker
- OpenMediaVault (if wanted) [here](https://www.openmediavault.org/)

## Getting Started
```
git pull https://github.com/tea-mo903/rpi-nas.git
cd rpi-nas
docker-compose up
```

## Components
- Heimdall Dashboard
- Plex
- Portainer
- CloudCmd
